Did you know that Tiptap offers resizable node views? This feature allows you to create custom node views that can be resized by the user. This is useful for creating resizable nodes like images or videos.

## Guide

### Create a basic resizable node view

To get started, you can use the `createResizableNodeView` function from the `@tiptap/core` package in your node's `addNodeView` method.

```js
import { Node, createResizableNodeView } from '@tiptap/core';

const MyNode = Node.create({
  name: 'myNode',
  group: 'block',

  addAttributes() {
    return {
      width: {
        default: null,
      },
      height: {
        default: null,
      },
    }
  },

  // add the resizable node view here
  addNodeView() {
    return ({ node, getPos, editor }) => {
      const el = document.createElement('div');
      el.style.position = 'relative';
      el.style.width = '100%';

      return createResizableNodeView({
        dom: el,
        editor,
        getPos,
        node,
        directions: {
          "top-right": true,
          "bottom-right": true,
          "top-left": true,
          "bottom-left": true,
        },
      })
    });
  },
});
```

### Customize the width and height attributes

By default, the `width` and `height` attributes on the node will be updated. If your node uses a different attribute name, you can customize the `widthAttribute` and `heightAttribute` options.

```js
createResizableNodeView({
  // ...
  widthAttribute: 'custom-width',
  heightAttribute: 'custom-height',
})
```

### Customize the directions

You have control over the directions that are available for resizing. By default, all four directions are enabled. You can customize this by passing an object with the directions you want to enable.

```js
createResizableNodeView({
  // ...
  directions: {
    "top": true,
    "bottom": true,
    "top": true,
    "bottom": true,
  },
})
```

### Preserve the aspect ratio

You can preserve the aspect ratio of the node by setting the `preserveAspectRatio` option to `true`.

```js
createResizableNodeView({
  // ...
  preserveAspectRatio: true,
})
```

## API

### Functions

#### `createResizableNodeView(options)`

Creates a resizable node view for the given node.

**Arguments**

- `options`: A [`ResizableNodeViewOptions`](#resizablenodeviewoptions) object.

### Types

#### `ResizableNodeViewOptions`

- `dom: HTMLElement`: The DOM element to use as the node view.
- `editor: Editor`: The editor instance.
- `getPos: () => number`: A function that returns the position of the node in the document.
- `node: Node`: The node to create the view for.
- `directions: ResizableNodeViewDirections`: An object with the directions that are available for resizing.
- `widthAttribute: string`: The attribute name to use for the width.
- `heightAttribute: string`: The attribute name to use for the height.
- `preserveAspectRatio: boolean`: Whether to preserve the aspect ratio of the node.
- `minWidth: number`: The minimum width of the node.
- `minHeight: number`: The minimum height of the node.
- `onUpdate: (node: Node, decorations: Decoration[], innerDecorations: Decoration[]) => boolean | undefined`: A callback function that is called when the node is updated.

#### `ResizableNodeViewDirections`

Can be one of the following:

- `"top"`
- `"bottom"`
- `"left"`
- `"right"`
- `"top-left"`
- `"top-right"`
- `"bottom-left"`
- `"bottom-right"`
