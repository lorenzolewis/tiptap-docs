The AI Toolkit lets you show AI-generated changes in a beautiful diff UI, so your users can review and accept or reject them.

First, configure the `reviewOptions` parameter:

```ts
// inside the useChat hook
const result = toolkit.executeTool({
  toolName,
  input
  currentChunk: currentChunk.current,
  reviewOptions: { mode: 'preview' },
})
```

## Styling suggestions

Create a `suggestions.css` file to style suggestions in red/green.

```css
.tiptap-ai-suggestion {
  background-color: oklch(0.637 0.237 25.331);
}

.tiptap-ai-suggestion--selected {
  background-color: oklch(0.505 0.213 27.518);
}

.tiptap-ai-suggestion-diff {
  background-color: oklch(0.723 0.219 149.579);
}

.tiptap-ai-suggestion-diff--selected {
  background-color: oklch(0.527 0.154 150.069);
}
```

Import the stylesheet in your app:

```ts
// Add this line to App.tsx
import './suggestions.css'
```

## Advanced review options

The `reviewOptions` parameter gives you full control over the review workflow and the UI.

If you want to let users prewiew changes before they're applied, set the `reviewOptions.mode` to `preview`. Instead, if you want to review changes after they're applied, set the `reviewOptions.mode` to `review`.

To render custom elements in the UI, set the `reviewOptions.displayOptions` parameter. There, you can set the `renderDecorations` option. It's a function that returns a list of ProseMirror decorations.

```ts
import { Decoration } from '@tiptap/pm/view'

// inside the useChat hook
const result = toolkit.executeTool({
  toolName,
  input,
  currentChunk: currentChunk.current,
  reviewOptions: {
    mode: 'preview',
    displayOptions: {
      renderDecorations: (options) => [
        ...options.getDefaultDecorations(),
        Decoration.widget(options.suggestion.range.to, () => {
          const element = document.createElement('span')
          element.textContent = 'âœ…'
          return element
        }),
      ],
    },
  },
})
```

You can see a complete reference of the `reviewOptions` parameter in the API reference for the [`executeTool` method](/content-ai/capabilities/ai-toolkit/primitives/execute-ai-agent-tools).
