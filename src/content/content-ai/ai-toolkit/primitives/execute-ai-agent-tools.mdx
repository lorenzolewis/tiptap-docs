Use `executeTool` to apply tool calls from your AI Agent to the editor.

Once you've added [tool definitions](/content-ai/ai-toolkit/tools/ai-sdk) to your AI Agent, the AI Agent will generate tool calls. Use the `executeTool` method to apply the tool calls to the editor.

## `executeTool`

Executes a supported tool by name and input.

### Parameters

- `options` (`ExecuteToolOptions`): Configuration for tool execution
  - `toolName` (`string`): Tool to execute. Common values: `insertContent`, `applyDiff`, `readFirstChunk`, `readNextChunk`, `readPreviousChunk`
  - `input` (`Record<string, any>`): Tool-specific input payload
  - `currentChunk` (`number`): Current chunk index when reading/editing chunked content
  - `reviewOptions?` (`ReviewOptions`): Control preview/review behavior
    - `mode?` (`'disabled' | 'review' | 'preview'`)
    - `showDiff?` (`boolean`)
    - `displayOptions?` (`DisplayOptions<{ suggestion: Suggestion }>`)

### Returns (`ExecuteToolResult`)

- `output` (`string`): Response message from the tool execution, to be read by the AI agent
- `hasError` (`boolean`): Whether there was an error during execution
- `currentChunk` (`number`): The current chunk index after execution, may be updated by some tools
- `unknownTool?` (`boolean`): Whether the tool is not recognized by the AI Toolkit

### Example

```ts
// Apply two diffs with preview mode and diff widgets
const result = toolkit.executeTool({
  toolName: 'applyDiff',
  input: {
    diffs: [{ context: '<p>Hello ', delete: 'world', insert: '<b>world</b>' }],
  },
  currentChunk: 0,
  reviewOptions: { mode: 'preview', showDiff: true },
})
```
