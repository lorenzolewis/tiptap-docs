Expose schema awareness so AI models understand your document structure.

## `getSchemaAwareness`

Returns a schema awareness string describing nodes, marks, attributes, and examples.

### Parameters

- `options?` (`GetSchemaAwarenessOptions`): Options for the `getSchemaAwareness` method
  - `customNodes?` (`SchemaAwarenessItem[]`): Custom schema awareness items to include in addition to the default ones. Defaults to `[]`

### Returns

- `string`: Human-readable schema awareness text suitable for system prompts

### Example

```ts
// Include a custom node in the awareness text
const awareness = toolkit.getSchemaAwareness({
  customNodes: [
    {
      extensionName: 'customNode',
      name: 'Custom Node',
      htmlTag: 'custom-element',
      description: 'A custom node type',
      attributes: [],
    },
  ],
})
```
